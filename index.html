git --version

check is there git on your computer and find out it verion

git config --global --list

find out settings of your GLOBAL settings for your git on your current computer

git config --global user.name

set your NAME for your git global setting on your computer

git config --global user.email

set your EMAIL for your git global setting on your computer

git init

turn current folder into repository and monitor the changes that occur there

git remote add origin { url repository }

create a connection between a local repository and a remote one by URL

git clone { url repository }

download all files of a remote repository and create a local copy of this (remote) repository

git status

show information about all uncommitted files in the current local repository

git add /some_files

prepares these files for the next commit by adding them to the git index

git add .

prepares all files for the next commit by adding them to the git index
